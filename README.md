# Website - Akmos Gestão Contábil

Este projeto consiste no desenvolvimento de um website institucional para a empresa Akmos Gestão Contábil. O objetivo do site é apresentar os serviços da empresa, facilitar o contato com clientes e disponibilizar algumas utilidades contábeis.

O site possui um layout moderno e responsivo, com seções organizadas para apresentar informações sobre a empresa, serviços oferecidos, avaliações de clientes e um formulário de contato.

## Funcionalidades

- Página inicial com apresentação da empresa
- Seção de serviços contábeis
- Área com avaliações de clientes
- Formulário de contato
- Página de utilidades com tabelas e links úteis
- Navegação simples e intuitiva

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi desenvolvido como trabalho freelance, com foco em criar um site institucional simples, funcional e visualmente agradável para apresentar os serviços da empresa na internet.
